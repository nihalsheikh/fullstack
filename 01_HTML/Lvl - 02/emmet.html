Emmets in HTML

01. For child emmet shortcut

use: > symbol in between tags

Example: nav>ul>li

output is as the following tags, see code

02. For Multiple tags emmet shortcut

use * symbol to get multiple tags

Example: ul>li*3

output is as following, see code

03. Item Numbering emmet shortcut

use dot symbol '.' for class attribute, followd by the nameScheme you want

with nameScheme write '$' dollar symbol in continuation

after '$' dollar symbol you can specify how many tags you want using '*' asterisk symbol

Example: ul>li.item$*3

Output is as follow, see code

04. ID emmet shortcut

To use ID as attribute in a tag, use the '#' symbol with tag followed by nameScheme

Example: ul#header

Output is as follows, see code
